Redrob AI Candidate Ranking

Data & AI Challenge

AI-powered candidate discovery and ranking system designed to identify the most relevant candidates from large talent pools.

Problem

Traditional ATS systems rely heavily on keyword matching and often fail to identify highly qualified candidates whose profiles use different terminology.

Solution

This project combines candidate profile analysis, skill extraction, experience evaluation, and AI-driven ranking to recommend the best-fit candidates.

Methodology

Final Score:

* Skills Match = 40%
* Experience Match = 30%
* Domain Relevance = 20%
* Education = 10%

Tech Stack

* Python
* Pandas
* NumPy
* Scikit-Learn

Output

A ranked candidate list generated in CSV/XLSX format.
